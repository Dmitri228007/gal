Здравствуйте, уважаемый пользователь спасибо что выбрали это приложение


Горячие клавиши:
#  Ctrl + f  <-- поиск альбома в галерее
#  Ctrl + h  <-- открытие окна с подсказками (этого окна)
#  Detete    <-- при просмотре альбома, удаляет просматриваемый альбом, 
при просмотре фотографии, удаляет просматриваемую фотографию
#  r         <-- (резервная кнопка, если что-то пошло не по плану)
обновляет спписок альбомов на главной странице 


Это небольшая памятка для удобства в использовании галереи:

Название альбома не должно начинаться и оканчиваться пробельными символами, 
в противном случае они будут удалены из названия, 
если название альбома будет пустой сторокой, будет ошибка: 
Пожалуйста, проверьте правильность\nнаписания названия
Название альбома должно быть уникальным, иначе будет ошибка: 
Не оригинальное название

Поиск по названию альбома предполагает, что вы знаете точное название искомого альбома,
в противном случае будет ошибка: альбом не найден

Добавление фотография подрузомевает следующие расширения фото: .jpeg; .png; .jpg; .gif
В противном случае файл не будет добавлено в альбом

К каждому фото можно оставлять комментарий (заметку).
Для сохранения комментария необходимо нажать на кнопку "Сохранить заметку"
Для удаления комментария необходимо нажать кнопку "Удалить заметку" и закрыть окно
